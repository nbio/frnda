HELP
====

> python pisa.py

REQUIREMENTS
============

- Reportlab Toolkit 2.1+
  <http://www.reportlab.org/>

- html5lib
  <http://code.google.com/p/html5lib/>

- pyPDF (optional)
  <http://pybrary.net/pyPdf/>

EXAMPLES
========

> python pisa.py -s test\test-loremipsum.html
> python pisa.py -s http://www.python.org
> python pisa.py test\test-*.html

PYTHON INTEGRATION
==================

Some simple demos of how to integrate PISA into
a Python program may be found here: test\simple.py
